# 数据库设计模板

## 任务拆分建议

### 大型任务："设计并实现完整的数据库表结构"

拆分为：
1. 设计 users 表
2. 设计 questions 表
3. 设计 constitution_results 表
4. 设计 foods 表
5. 设计 recipes 表
6. 编写迁移脚本
7. 创建索引
8. 编写种子数据脚本

### 估算时间
- 总计: 2-3 小时
- 每个子任务: 10-20 分钟

### 验收标准
- [ ] 表结构符合 MVP 需求
- [ ] 外键关系正确
- [ ] 索引创建完成
- [ ] 迁移脚本可重复执行
- [ ] 种子数据正确导入
