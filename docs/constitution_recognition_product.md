### 2.2 CCMQ量表说明

**CCMQ（Constitution in Chinese Medicine Questionnaire）**是中医体质辨识的金标准：

| 维度 | 传统版 | 本产品精简版 |
|-----|-------|-------------|
| 题目数量 | 60题 | 30题 |
| 完成时间 | 15-20分钟 | 5-8分钟 |
| 评分方式 | 5分制Likert量表 | 5分制Likert量表 |
| 判定阈值 | 主要体质≥40分，次要体质≥30分 | 保持一致 |
| 准确率 | 75-85% | 目标85%+ |

### 2.3 体质判定算法

```
┌─────────────────────────────────────────────────────────────────────┐
│                        体质识别流程                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  第一步：用户作答                                                    │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ 题目：您容易疲乏吗？                                          │   │
│  │ 选项：①没有 ②很少 ③有时 ④经常 ⑤总是                         │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  第二步：计算各维度原始分                                            │
│  每个体质维度包含若干相关问题，按题目归属累加得分                      │
│                              ↓                                      │
│  第三步：转换为百分制                                                │
│  转换分数 = 原始分 × 2.5  （100分制）                                │
│                              ↓                                      │
│  第四步：判定体质类型                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ 判定规则：                                                   │   │
│  │ • 平和质判定：平和质分数 ≥ 60分，且其他8种体质分数 < 40分      │   │
│  │ • 偏颇体质判定：某体质分数 ≥ 40分 → 主要体质                  │   │
│  │ • 兼夹体质判定：多个体质分数 ≥ 30分 → 次要体质                │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                              ↓                                      │
│  第五步：生成体质报告                                                │
│  主要体质 + 次要体质（如有） + 个性化养生方案                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 3.2 体质测试功能

#### 功能描述

用户通过30题精简问卷，在5-8分钟内完成体质测试，系统自动计算并判定用户的体质类型。

#### 问卷设计

| 维度 | 题目数量 | 覆盖内容 |
|-----|---------|---------|
| 平和质 | 4题 | 精力、睡眠、胃口、适应力 |
| 气虚质 | 4题 | 疲乏、气短、自汗、感冒 |
| 阳虚质 | 4题 | 手足不温、喜热饮、畏寒 |
| 阴虚质 | 4题 | 口干、咽燥、盗汗、手足心热 |
| 痰湿质 | 3题 | 体形、身重、面部油腻 |
| 湿热质 | 3题 | 面垢、口苦、大便黏滞 |
| 血瘀质 | 3题 | 面色、瘀斑、疼痛 |
| 气郁质 | 3题 | 情绪、胸胁、叹气 |
| 特禀质 | 2题 | 过敏、喷嚏、哮喘 |

#### 用户交互流程

```
开始测试
   │
   ├─→ 显示测试说明（预计耗时、隐私说明）
   │
   ├─→ 逐题展示（支持进度保存）
   │    │
   │    ├─→ 题目：[大字清晰显示]
   │    │
   │    └─→ 选项：①没有 ②很少 ③有时 ④经常 ⑤总是
   │
   ├─→ 提交答卷
   │
   ├─→ 计算分析（后台计算各维度得分）
   │
   └─→ 生成报告
```

### 3.5 运动推荐功能

#### 体质与运动匹配

| 体质类型 | 推荐运动 | 避免运动 | 运动原则 |
|---------|---------|---------|---------|
| 平和质 | 各类运动均可 | - | 适量运动，保持健康 |
| 气虚质 | 八段锦、太极拳、散步 | 剧烈运动、大汗淋漓 | 量力而行，微微出汗 |
| 阳虚质 | 慢跑、快走、瑜伽 | 游泳、冷水运动 | 动静结合，注意保暖 |
| 阴虚质 | 散步、太极、气功 | 高温环境运动 | 适量运动，及时补水 |
| 痰湿质 | 有氧运动、跑步、游泳 | 久坐少动 | 持续运动，强度递增 |
| 湿热质 | 游泳、跑步、球类 | 高温环境运动 | 强度适中，及时补水 |
| 血瘀质 | 太极、舞蹈、瑜伽 | 久坐不动 | 循序渐进，持之以恒 |
| 气郁质 | 太极、瑜伽、舞蹈 | 剧烈竞技 | 动静结合，调节情绪 |
| 特禀质 | 太极、散步、瑜伽 | 接触过敏原运动 | 避免过敏，温和运动 |

#### 运动方案示例

```
【气虚质运动方案】

目标：增强体质，提升免疫力
原则：柔和为主，微微出汗，避免过度

推荐运动：
1. 八段锦（每日15-20分钟）
   - 第一式：双手托天理三焦
   - 第二式：左右开弓似射雕
   - ...

2. 太极拳（每周3-4次，每次20-30分钟）
   - 简化24式太极拳
   - 动作缓慢连贯，呼吸自然

3. 散步（每日30分钟）
   - 饭后1小时进行
   - 步速适中，每分钟80-100步

运动时间：上午9-10点最佳
注意事项：
• 运动后及时擦汗，避免受风
• 感到疲劳时立即休息
• 空腹或过饱时不宜运动
```

#### 二十四节气养生

```
   │            │             │              │             │

```

#### 季节养生内容示例

```
【春季养生 - 气虚质】

季节特点：春生万物，阳气升发，肝气旺盛

养生重点：
• 养阳：适当晒太阳，多做户外活动
• 护肝：保持心情舒畅，避免情绪波动
• 健脾：春天肝旺脾弱，需健脾防肝克

饮食建议：
• 宜吃：韭菜、菠菜、山药、大枣、蜂蜜
• 忌吃：酸味过多（肝旺克脾）、生冷食物
• 推荐食谱：韭菜炒鸡蛋、山药红枣粥

运动建议：
• 推荐：踏青、放风筝、太极拳
• 时间：上午9-10点，阳光充足时
• 强度：循序渐进，微微出汗

起居建议：
• 作息：晚睡早起（22:00入睡，6:00起床）
```

### 4.2 核心算法

#### 体质判定算法（Python示例）

```python
class ConstitutionService:
    """体质分析服务"""

    SCORE_CONVERT_FACTOR = 2.5  # 转换为百分制
    THRESHOLD_PRIMARY = 40      # 主要体质阈值
    THRESHOLD_SECONDARY = 30    # 次要体质阈值
    THRESHOLD_PEACE = 60        # 平和质阈值

    def analyze(self, answers: List[int], questions: List[Dict]) -> Dict:
        """分析体质类型"""
        # 计算各维度得分
        scores = self._calculate_scores(answers, questions)

        # 判定体质类型
        result = self._determine_constitution(scores)

        # 生成报告
        report = self._generate_report(result)

        return {
            "primary": result["primary"],
            "secondary": result["secondary"],
            "scores": scores,
            "report": report
        }

    def _calculate_scores(self, answers, questions):
        """计算各体质维度得分"""
        scores = {
            "peace": 0.0, "qi_deficiency": 0.0, "yang_deficiency": 0.0,
            "yin_deficiency": 0.0, "phlegm_damp": 0.0, "damp_heat": 0.0,
            "blood_stasis": 0.0, "qi_depression": 0.0, "special": 0.0
        }

        for i, question in enumerate(questions):
            ctype = question.get("constitution_type")
            answer = answers[i] if i < len(answers) else 0
            scores[ctype] += answer

        # 转换为百分制
        for key in scores:
            scores[key] = min(100, scores[key] * self.SCORE_CONVERT_FACTOR)

        return scores

    def _determine_constitution(self, scores):
        """判定体质类型"""
        result_types = []
        max_score = 0
        primary_type = ""

        for ctype, score in scores.items():
            if score >= self.THRESHOLD_SECONDARY:
                result_types.append({"type": ctype, "score": score})
                if score > max_score:
                    max_score = score
                    primary_type = ctype

        # 平和质判定
        if scores["peace"] >= self.THRESHOLD_PEACE:
            primary_type = "peace"

        return {
            "primary": primary_type,
            "secondary": [r for r in result_types if r["type"] != primary_type]
        }
```

#### 推荐算法（跨产品推荐）

```python
class RecommendationService:
    """跨产品推荐引擎"""

    def __init__(self):
        self.weight = {
            "constitution_match": 0.4,  # 体质匹配度
            "season_bonus": 0.2,         # 季节加成
            "user_preference": 0.2,      # 用户偏好
            "popularity": 0.1,           # 热度
            "feedback_score": 0.1        # 历史反馈
        }

    async def recommend_diet(self, user_id: str) -> List[Dict]:
        """推荐饮食方案"""
        # 1. 获取用户画像
        profile = await self._get_user_profile(user_id)

        # 2. 获取候选食物池
        candidates = await self._get_food_candidates()

        # 3. 计算推荐分数
        scored_items = []
        for food in candidates:
            score = await self._calculate_score(food, profile)
            scored_items.append({**food, "recommend_score": score})

        # 4. 排序返回Top-N
        scored_items.sort(key=lambda x: x["recommend_score"], reverse=True)
        return scored_items[:20]
```
