## 1. 食材库详解

### 1.1 谷物类食材

#### 1.1.1 山药（淮山）

| 属性 | 内容 |
|------|------|
| **中文名称** | 山药 |
| **别名** | 怀山药、淮山、薯蓣、土薯 |
| **类别** | 谷物 |
| **性味** | 性平，味甘 |
| **归经** | 脾、肺、肾经 |
| **适用体质** | 气虚质、阴虚质、平和质 |
| **禁忌体质** | 湿热质、痰湿质（湿盛者） |
| **功效** | 健脾养胃、补肺益肾、固精止带 |
| **营养成分** | 含黏液蛋白、维生素、淀粉酶、多酚氧化酶等 |
| **每日用量** | 50-100g |
| **最佳食用时间** | 早晚餐 |
| **食用方法** | 蒸、煮、炖、炒 |
| **宜配食材** | 莲子、枸杞、红枣、茯苓 |
| **忌配食材** | 碱性食物（同食会破坏黏液蛋白） |
| **注意事项** | 湿盛者慎用，生山药有毒需煮熟 |

**详细说明：**

山药是中医食疗中的经典补品，被誉为"补脾第一要药"。其性平和，不燥不腻，是气虚质、阴虚质人群的理想食材。

山药中的黏液蛋白能保护胃黏膜，具有健脾养胃的功效；同时含有丰富的淀粉酶和多酚氧化酶，有助于消化吸收。

对于气虚质人群，山药能补中益气，增强体质；对于阴虚质人群，山药能滋阴润燥，生津止渴。

**经典搭配：**
- 山药 + 红枣：健脾益气，适合气血两虚
- 山药 + 枸杞：滋补肝肾，适合肝肾阴虚
- 山药 + 莲子：健脾止泻，适合脾虚泄泻
- 山药 + 芡实：健脾固肾，适合肾虚遗精

| 分类 | 食材 | 用量 |

| 调料 | 冰糖 | 适量 |

## 3. 体质调理方案

### 3.1 气虚质调理方案

**体质特征：**
- 元气不足，疲乏无力
- 气短懒言，易出汗
- 面色苍白，食欲不振
- 易感冒，恢复慢

**调理原则：** 补气健脾，增强体质

**推荐食材：**
- 谷物类：山药、小米、糯米、芡实、莲子
- 肉类：鸡肉、牛肉、羊肉
- 药材类：黄芪、党参、红枣、桂圆
- 水果类：苹果、樱桃

**禁忌食材：**
- 性寒凉食物：绿豆、苦瓜、西瓜
- 破气食物：萝卜、大蒜、山楂

**推荐食谱：**
1. 山药红枣粥
2. 黄芪炖鸡汤
3. 人参莲子粥
4. 红枣桂圆茶

**生活建议：**
- 避免过度劳累
- 保证充足睡眠
- 适度运动，如散步、太极
- 保持心情舒畅

---

### 3.2 阳虚质调理方案

**体质特征：**
- 阳气不足，畏寒怕冷
- 手足不温，喜热饮
- 面色苍白，精神不振
- 腰膝酸软，小便清长

**调理原则：** 温阳散寒，温暖身体

**推荐食材：**
- 谷物类：糯米、羊肉、韭菜
- 肉类：羊肉、牛肉、鸡肉
- 药材类：肉桂、干姜、当归
- 调味品：生姜、葱、蒜

**禁忌食材：**
- 性寒凉食物：绿豆、苦瓜、西瓜
- 生冷食物：冰饮、生鱼片

**推荐食谱：**
1. 当归生姜羊肉汤
2. 韭菜炒鸡蛋
3. 羊肉萝卜汤
4. 生姜红糖茶

**生活建议：**
- 注意保暖，避免受凉
- 适当增加运动
- 规律作息，避免熬夜
- 保持心情开朗

---

### 3.3 阴虚质调理方案

**体质特征：**
- 阴液亏虚，口燥咽干
- 手足心热，盗汗
- 面色潮红，心烦易怒
- 大便干燥，小便短赤

**调理原则：** 滋阴润燥，清热生津

**推荐食材：**
- 谷物类：小米、山药、百合
- 水果类：梨、西瓜、柚子、猕猴桃
- 蔬菜类：菠菜、番茄、冬瓜
- 药材类：百合、银耳、枸杞、麦冬

**禁忌食材：**
- 性温燥食物：羊肉、辣椒、生姜
- 辛辣刺激：辣椒、花椒、胡椒

**推荐食谱：**
1. 百合银耳羹
2. 梨膏糖
3. 小米百合粥
4. 枸杞菊花茶

**生活建议：**
- 避免熬夜
- 保持心情舒畅
- 避免剧烈运动
- 多饮水

### 3.4 痰湿质调理方案

**体质特征：**
- 痰湿内盛，形体肥胖
- 面部油脂多，口黏腻
- 胸闷痰多，身重不爽
- 大便黏腻，小便混浊

- 谷物类：薏米、红豆、芡实

**禁忌食材：**
- 滋腻食物：蜂蜜、红枣、桂圆
- 甜食：糖果、蛋糕
- 油腻食物：肥肉、油炸食品

**推荐食谱：**
1. 薏米红豆粥
2. 冬瓜薏米汤
3. 陈皮茯苓茶
4. 薏米炒白术

**生活建议：**
- 控制饮食，避免暴饮暴食
- 加强运动
- 保持居住环境干燥
- 避免潮湿环境

---

### 3.5 湿热质调理方案

**体质特征：**
- 湿热内蕴，面垢油光
- 易生痤疮，口苦口臭
- 大便黏滞，小便短赤
- 舌质红，苔黄腻

**禁忌食材：**
- 性温热食物：羊肉、辣椒、生姜
- 油腻食物：肥肉、油炸食品
- 酒类：白酒、啤酒

- 适度运动
