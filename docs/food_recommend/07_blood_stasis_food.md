# 血瘀质饮食推荐

## 体质概述

血瘀质是指血行不畅，以肤色晦黯、舌质紫黯为主要特征的体质类型。主要表现为肤色晦黯、色素沉着、容易出现瘀斑、口唇黯淡或紫、舌质黯淡或有瘀点等。

## ❌ 不宜食物

| 食物 | 原因 |
|------|------|
| **柿子** | 性寒涩，易致血行不畅 |
| **石榴** | 性收敛，不利于活血 |
| **乌梅** | 性收敛，不利于血行 |
| **酸梅** | 性收敛，应少食 |
| **李子** | 性涩，不利于活血 |
| **柠檬** | 性酸，不宜多吃 |
| **肥肉** | 油腻滋腻，阻碍血行 |
| **奶油** | 油腻滋腻，加重血瘀 |
| **甜食** | 滋腻碍血行 |
| **冷饮** | 寒凝血脉，应避免 |

## 🍳 推荐食谱

### 1. 山楂玫瑰花茶
**配料**：山楂 10g，玫瑰花 5g，冰糖 适量
**做法**：
1. 山楂、玫瑰花洗净
2. 用沸水冲泡，焖 20 分钟
3. 加冰糖调味即可
**功效**：活血化瘀，疏肝解郁

### 2. 黑木耳炒肉
**配料**：黑木耳 50g，猪肉 200g，生姜 10g
**做法**：
1. 黑木耳泡发撕成小朵
2. 猪肉切片，生姜切丝
3. 热锅爆香姜丝，下肉片炒至变色
4. 加入黑木耳炒熟，调味
**功效**：益气活血，营养美味

### 3. 黑豆粥
**配料**：黑豆 50g，大米 100g，大枣 5 枚
**做法**：
1. 黑豆提前浸泡
2. 大米洗净
3. 所有材料同煮成粥
**功效**：活血利水，健脾益胃

### 4. 当归生姜羊肉汤
**配料**：羊肉 500g，当归 15g，生姜 30g
**做法**：
1. 羊肉切块焯水
2. 当归、生姜洗净
3. 所有材料放入砂锅，加水炖煮 2 小时
**功效**：活血散寒，温经止痛

### 5. 洋葱炒鸡蛋
**配料**：洋葱 1 个，鸡蛋 3 个
**做法**：
1. 洋葱切丝，鸡蛋打散
2. 热锅炒鸡蛋盛起
3. 再炒洋葱，合并翻炒，调味
**功效**：理气活血，简单营养

### 6. 桃仁粥
**配料**：桃仁 15g，大米 100g，红糖 适量
**做法**：
1. 桃仁去皮炒熟
2. 大米洗净
3. 所有材料同煮成粥，加红糖调味
**功效**：活血祛瘀，润肠通便

## 💡 饮食原则

1. **活血化瘀为主**
   - 多食用活血化瘀的食物
   - 如山楂、玫瑰花、黑豆等

2. **温通经脉**
   - 适量食用温热性质的食物
   - 如生姜、洋葱、当归等

3. **避免寒涩收敛**
   - 不宜食用柿子、石榴等
   - 少食酸味过重的食物

4. **适量食用醋**
   - 适量食醋有助于活血
   - 可用于调味或醋泡食物

5. **多食黑色食物**
   - 黑色食物多能活血
   - 如黑豆、黑木耳等

### 红花茶

**做法**：用沸水冲泡，焖 20 分钟即可
**功效**：活血通经，祛瘀止痛

**配料**：生姜 3 片，大枣 3 枚，红糖 适量

**功效**：温经活血，补气养血

## ⚠️ 注意事项

- 血瘀质者应注意保暖
- 避免受寒，寒邪易致血行不畅
- 女性经期注意调养
- 保持适量运动，促进血液循环
- 避免久坐，定时活动

---

## 🏃 运动建议

1. **有氧运动**
   - 跑步、跳绳、游泳
   - 促进血液循环

2. **力量训练**
   - 适当力量训练
   - 增强肌肉力量

3. **伸展运动**
   - 瑜伽、太极、八段锦
   - 拉伸筋骨，促进血行

4. **按摩推拿**
   - 定期按摩推拿
   - 疏通经络，促进血行

---

## 🌡️ 四季调养

- **春季**：食用洋葱、芹菜等温通食物
- **夏季**：食用黑木耳、茄子等平和食物
- **秋季**：食用山楂、葡萄等活血食物
- **冬季**：最佳温通时节，多食用羊肉、当归等

---

## 🥗 养生建议

1. **起居调养**
   - 注意保暖，避免受寒
   - 不宜久坐，定时活动
   - 保持规律作息

2. **情志调养**
   - 保持心情愉快
   - 避免抑郁不乐
   - 适当娱乐放松

3. **按摩调养**
   - 按摩血海、三阴交等穴位
   - 促进血液循环
   - 每日 10-15 分钟

## 🌿 外用调养

### 生姜泡脚
**配料**：生姜 50g，艾叶 30g
**做法**：
1. 生姜切片，艾叶洗净
2. 煮水后兑入温水
3. 泡脚 20 分钟
**功效**：温经通络，活血散寒

### 红花泡脚
**配料**：红花 15g，川芎 15g
**做法**：
1. 煮水后兑入温水
2. 泡脚 20 分钟
**功效**：活血通络，祛瘀止痛
