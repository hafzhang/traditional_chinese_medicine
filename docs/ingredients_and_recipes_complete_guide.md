### 1.1 谷物类食材（60种）

#### 1.1.1 山药（淮山）

| 属性 | 内容 |
|------|------|
| **中文名称** | 山药 |
| **别名** | 怀山药、淮山、薯蓣、土薯 |
| **类别** | 谷物 |
| **性味** | 性平，味甘 |
| **归经** | 脾、肺、肾经 |
| **适用体质** | 气虚质、阴虚质、平和质 |
| **禁忌体质** | 湿热质、痰湿质（湿盛者） |
| **功效** | 健脾养胃、补肺益肾、固精止带 |
| **营养成分** | 含黏液蛋白、维生素、淀粉酶、多酚氧化酶等 |
| **每日用量** | 50-100g |
| **最佳食用时间** | 早晚餐 |
| **食用方法** | 蒸、煮、炖、炒 |
| **宜配食材** | 莲子、枸杞、红枣、茯苓 |
| **忌配食材** | 碱性食物（同食会破坏黏液蛋白） |
| **注意事项** | 湿盛者慎用，生山药有毒需煮熟 |
| **热量(100g)** | 56 kcal |
| **蛋白质** | 1.9g |
| **脂肪** | 0.2g |
| **碳水化合物** | 11.6g |
| **膳食纤维** | 0.8g |

**营养价值分析：**

山药富含黏液蛋白，能保护胃黏膜，具有健脾养胃的功效。山药中的淀粉酶和多酚氧化酶有助于消化吸收。

山药含有多种维生素和矿物质，包括维生素C、维生素B6、钾、锰等。山药中的薯蓣皂苷有调节内分泌的作用。

山药的热量较低，适合减肥人群食用。山药的升糖指数较低，糖尿病患者适量食用有益。

**经典搭配：**
- 山药 + 红枣：健脾益气，适合气血两虚
- 山药 + 枸杞：滋补肝肾，适合肝肾阴虚
- 山药 + 莲子：健脾止泻，适合脾虚泄泻
- 山药 + 芡实：健脾固肾，适合肾虚遗精
- 山药 + 鸡肉：补中益气，适合体虚乏力

| **禁忌体质** | 胃寒质（慎用） |
| **功效** | 健脾和胃、补虚损、安神助眠 |
| **营养成分** | 含蛋白质、脂肪、碳水化合物、钙、磷、铁、维生素B1等 |
| **每日用量** | 50-100g |
| **最佳食用时间** | 早餐、晚餐 |
| **食用方法** | 煮粥、蒸饭 |
| **宜配食材** | 红枣、桂圆、莲子、百合 |
| **忌配食材** | 杏仁（同食不利消化） |
| **注意事项** | 性凉，胃寒者可与生姜搭配 |
| **热量(100g)** | 358 kcal |
| **蛋白质** | 9.0g |
| **脂肪** | 3.1g |
| **碳水化合物** | 73.5g |
| **膳食纤维** | 1.6g |

**营养价值分析：**

小米富含色氨酸，有助于改善睡眠。小米含有的维生素B1能促进消化液分泌，增进食欲。

小米中的铁含量较高，有助于预防贫血。小米中的锌元素有助于增强免疫力。

小米粥易于消化，适合老人、儿童、病人食用。小米有"代参汤"之称，是产妇的理想食品。

**经典搭配：**
- 小米 + 红枣：补血安神，适合失眠多梦
- 小米 + 百合：滋阴润肺，适合肺燥咳嗽
- 小米 + 山药：健脾养胃，适合脾胃虚弱
- 小米 + 桂圆：补气养血，适合气血两虚
- 小米 + 南瓜：养胃健脾，适合消化不良

| 属性 | 内容 |

| **类型** | 粥类 |
| **难度** | 简单 |
| **烹饪时间** | 45分钟 |
| **份量** | 2人份 |
| **适用体质** | 气虚质、阴虚质 |
| **主治症状** | 食欲不振、疲劳乏力、消化不良 |
| **适用季节** | 春、秋、冬 |
| **功效** | 健脾养胃、补肺益气 |
| **健康益处** | 适合脾胃虚弱者，增强消化功能 |

**食材清单：**

| 分类 | 食材 | 用量 |
|------|------|------|
| 主料 | 山药 | 100g |
| 主料 | 大米 | 100g |
| 辅料 | 红枣 | 10枚 |
| 调料 | 冰糖 | 适量 |

**制作步骤：**

1. 山药去皮洗净，切成小块
2. 大米淘洗干净，红枣去核洗净
3. 锅中加入适量清水，放入大米
4. 大火煮开后转小火煮20分钟
5. 加入山药块和红枣，继续煮15分钟
6. 加入适量冰糖调味，煮至米烂粥稠即可

**注意事项：**
- 山药去皮后易氧化变黑，切好后可浸泡在水中
- 糖尿病患者可不加冰糖或改用代糖
- 煮粥时需不时搅拌，防止粘锅

**营养分析（每100g）：**
- 热量：85 kcal
- 蛋白质：2.1g
- 脂肪：0.3g
- 碳水化合物：18.5g
- 膳食纤维：0.9g

**功效说明：**
山药红枣粥健脾养胃，适合脾胃虚弱、食欲不振的人群。
