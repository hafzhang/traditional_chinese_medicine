## 五、交互设计

### 5.1 点击反馈

- **按下效果**：卡片缩小至 98%
- **持续时间**：300ms
- **效果**：`transform: scale(0.98)`

### 5.2 导航逻辑

```javascript
function navigateTo(path) {
  // 判断是否为 tabBar 页面
  const tabBarPages = ['/pages/test/test', '/pages/food/list', '/pages/index/index']

  if (tabBarPages.includes(path)) {
    uni.switchTab({ url: path })
  } else {
    uni.navigateTo({ url: path })
  }
}
```

### 5.3 特殊处理

- 体质测试：使用 `switchTab`（tabBar 页面）
- 其他功能：使用 `navigateTo`（普通页面）
- 导航失败：显示"页面开发中"提示

## 八、实现要点

### 8.1 技术栈

- **框架：** Vue 3 + Composition API
- **UI：** uni-app
- **样式：** SCSS
- **导航：** uni-app 路由 API

### 8.2 关键代码结构

```vue
<template>
  <view class="page-container">
    <!-- Hero 区域 -->
    <view class="hero-section">...</view>

    <!-- 功能导航区域 -->
    <view class="features-section">
      <view class="section-title">功能导航</view>
      <view class="features-grid">
        <view v-for="item in features" @click="navigateTo(item.path)">
          <!-- 功能卡片 -->
        </view>
      </view>
    </view>

    <!-- 底部信息区域 -->
    <view class="footer-section">...</view>
  </view>
</template>

<script setup>
import { ref } from 'vue'

// 功能列表数据
const features = ref([...])

// 导航函数
function navigateTo(path) { ... }
</script>

<style lang="scss" scoped>
// 样式定义
</style>
```

### 8.3 注意事项

1. **TabBar 页面处理**：体质测试在 tabBar 中，需要用 `switchTab`
2. **渐变背景**：使用 inline style 动态设置渐变色
3. **点击反馈**：添加 `:active` 伪类实现点击缩放效果
4. **图标居中**：使用 flex 布局确保图标在渐变背景中居中
