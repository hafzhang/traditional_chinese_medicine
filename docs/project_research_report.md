# 中医体质养生助手项目调查报告

## 项目概述

**项目名称**：中医体质养生助手
**项目类型**：健康管理 / 中医智慧医疗
**核心功能**：用户通过问卷识别自身体质类型，获取个性化养生方案

---

## 一、理论基础

### 1.1 中医九种体质理论

本项目基于国医大师、中国工程院院士**王琦教授**提出的"中医九种体质分类"理论。该理论历经30余年研究，对中国5大地域9省26市2万多例病人进行流行病学调查得出，已被纳入国家标准。

| 体质类型 | 总体特征 | 常见表现 | 发病倾向 | 占比 |
|---------|---------|---------|---------|------|
| **平和质** | 阴阳气血调和 | 面色红润、精力充沛、睡眠良好、二便正常 | - | 约32.75% |
| **气虚质** | 元气不足，气息低弱 | 疲乏气短、易出汗、容易感冒、肌肉松软 | 易患感冒、内脏下垂 | - |
| **阳虚质** | 阳气不足，畏寒怕冷 | 手足不温、喜热饮食、畏寒怕冷 | 易患水肿、腹泻等寒性疾病 | - |
| **阴虚质** | 阴液亏少，口燥咽干 | 口燥咽干、手足心热、易心烦失眠、大便干燥 | 易患便秘、甲亢等疾病 | - |
| **痰湿质** | 痰湿凝聚，体型肥胖 | 体型肥胖、腹部肥满、口黏苔腻 | 易患高血压、糖尿病、高脂血症 | - |
| **湿热质** | 湿热内蕴，容易长痘 | 面垢油光、易生痤疮、口苦口臭、大便黏滞 | 易患皮肤疾病、泌尿系统感染 | - |
| **血瘀质** | 血行不畅，易有瘀斑 | 面色晦暗、易有瘀斑、女性多见痛经 | 易患肿瘤、冠心病 | - |
| **气郁质** | 气机郁滞，情绪不畅 | 闷闷不乐、情绪低沉、多愁善感、乳房胀痛 | 易患抑郁症、失眠 | - |
| **特禀质** | 先天失常，易过敏 | 过敏体质、容易哮喘、鼻塞打喷嚏、皮肤起荨麻疹 | 易患过敏性鼻炎、哮喘、湿疹 | - |

## 二、问卷设计标准

### 2.1 标准量表：CCMQ（中医体质量表）

由王琦院士和朱燕波教授团队开发，是目前应用最广泛的中医体质辨识工具。

#### 版本演进
- **60条目原版**：完整版本
- **41条目简短版**：优化简化版本
- **30条目版本**：精简版本（CCMQ-30）

#### 计分方式
采用5点Likert计分法：
- 没有（1分）
- 很少（2分）
- 有时（3分）
- 常常（4分）
- 总是（5分）

#### 题目开发流程
```
研究目的确定 → 体质类型概念框架建立 → 条目收集和条目库形成
→ 条目精选 → 问题形成 → 预调查 → 调查实施 → 信度效度评价
```

#### 统计学筛选标准
| 筛选方法 | 用途 |
|---------|------|
| 变异度法 | 评估题目区分度 |
| 条-总相关系数法 | 评估题目与总分的相关性 |
| 克朗巴赫α系数法 | 评估内部一致性信度 |
| 因子分析法 | 评估结构效度 |
| 临界比率法 | 评估题目鉴别力 |

## 三、技术实现方案

### 3.1 体质识别算法

根据最新研究（2024-2025），可采用以下算法：

#### 方案一：传统评分法（推荐首选）
- 基于CCMQ量表计算各维度得分
- 按照判定标准转换成百分制
- 判定偏颇体质（≥40分且平和质<60分）
- 优点：准确可靠、可解释性强

#### 方案二：机器学习方法
- **Random Forest**（随机森林）：被证明为最优模型
- **MLP**（多层感知机）
- **LSTM**（长短期记忆网络）
- **DBN**（深度信念网络）

## 四、数据库设计

### 4.1 核心数据表

```sql
-- 用户表
CREATE TABLE users (
    id UUID PRIMARY KEY,
    username VARCHAR(50) UNIQUE,
    email VARCHAR(100) UNIQUE,
    password_hash VARCHAR(255),
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);

-- 问卷表
CREATE TABLE questionnaires (
    id UUID PRIMARY KEY,
    title VARCHAR(200),
    description TEXT,
    version VARCHAR(20),
    total_questions INT,
    created_at TIMESTAMP
);

-- 问题表
CREATE TABLE questions (
    id UUID PRIMARY KEY,
    questionnaire_id UUID REFERENCES questionnaires(id),
    question_text TEXT,
    question_type VARCHAR(20), -- single_choice, multiple_choice, scale
    constitution_type VARCHAR(50), -- 关联的体质类型
    order_index INT,
    created_at TIMESTAMP
);

-- 用户问卷回答表
CREATE TABLE user_responses (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    questionnaire_id UUID REFERENCES questionnaires(id),
    answers JSONB, -- 存储所有问题的答案
    submitted_at TIMESTAMP,
    created_at TIMESTAMP
);

-- 体质识别结果表
CREATE TABLE constitution_results (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    response_id UUID REFERENCES user_responses(id),
    primary_constitution VARCHAR(50), -- 主要体质
    secondary_constituitions JSONB, -- 兼夹体质
    scores JSONB, -- 各体质维度得分
    algorithm_used VARCHAR(100),
    created_at TIMESTAMP
);

-- 体质知识库表
CREATE TABLE constitution_knowledge (
    id UUID PRIMARY KEY,
    constitution_type VARCHAR(50) UNIQUE, -- 体质类型
    description TEXT,
    characteristics TEXT,
    dietary_advice JSONB, -- 饮食建议
    exercise_advice JSONB, -- 运动建议
    lifestyle_advice JSONB, -- 作息建议
    seasonal_advice JSONB, -- 季节养生建议
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);

-- 食物库表
CREATE TABLE foods (
    id UUID PRIMARY KEY,
    name VARCHAR(100),
    nature VARCHAR(20), -- 寒、凉、平、温、热
    flavor VARCHAR(50), -- 酸、苦、甘、辛、咸
    meridian TEXT[], -- 归经
    nutrition_info JSONB,
   禁忌_体质 VARCHAR(50)[], -- 不适合的体质
    适用_体质 VARCHAR(50)[], -- 适合的体质
    created_at TIMESTAMP
);

-- 推荐记录表
CREATE TABLE recommendations (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    constitution_result_id UUID REFERENCES constitution_results(id),
    recommendation_type VARCHAR(50), -- diet, exercise, lifestyle, seasonal
    content JSONB,
    created_at TIMESTAMP
);
```

## 五、个性化推荐方案

### 5.1 饮食推荐

#### 气虚质
- **宜吃**：糯米、粳米、小麦、红薯、牛肉、大枣、桂圆、蜂蜜、小米、银耳、豆腐、鸡肉
- **忌吃**：山楂、佛手柑、槟榔、大蒜、苤蓝、萝卜缨、香菜、芜菁

#### 阳虚质
- **宜吃**：性质平和、偏温热、易消化的食物（大枣、红糖、鸡蛋、龙眼）
- **忌吃**：肥甘油腻、生冷苦寒的食物

#### 阴虚质
- **宜吃**：绿豆、乌贼、甲鱼、海参、螃蟹、芹菜、香蕉、冬瓜、菊花
- **忌吃**：辛辣刺激性食物

#### 痰湿质
- **宜吃**：薏米、小麦、小米、鸭肉、猪肉、鸡蛋、莲藕、黄瓜、芹菜、西瓜、苹果、梨
- **忌吃**：温热辛辣食物、生冷食物、辛辣、肥甘油腻的食物

#### 湿热质
- **宜吃**：清淡甘平易吸收之食物（芭乐、苹果、葡萄）
- **忌吃**：寒涼冰冷食物（冰品、茶类饮品、西瓜、水梨）、刺激性食物

#### 血瘀质
- **宜吃**：活血祛瘀食物（山楂、桃仁、陈皮、玫瑰花、佛手）、经常吃醋

### 5.2 运动推荐

| 体质 | 运动强度 | 推荐运动 |
|-----|----------|----------|
| 气虚 | 低强度 | 散步、八段锦、五禽戏、跳绳、舞蹈 |
| 阳虚 | 温和运动 | 太极拳、八段锦 |
| 阴虚 | 低强度 | 散步、调息养神 |
| 痰湿 | 中等强度 | 快走、游泳、有氧运动 |
| 湿热 | 中等强度 | 慢跑、羽毛球、乒乓球 |
| 血瘀 | 中等强度 | 快走、太极拳、舞蹈 |
| 气郁 | 中等强度 | 瑜伽、太极、户外运动 |

### 5.3 季节养生建议

| 季节 | 养生原则 | 重点 | 脏器对应 |
|-----|---------|-----|---------|
| **春季** | 养肝息风、养阳防风 | 生发阳气，保持心情愉快 | 肝属木 |
| **夏季** | 养心安神、保健脾胃 | 保护阳气，神清气和 | 心属火 |
| **秋季** | 养阴润燥 | 万物内敛，适合养阴 | 肺属金 |
| **冬季** | 养阴补肾、藏阳御寒 | 补肾精，养阴为主 | 肾属水 |

## 八、技术难点与解决方案

### 8.1 体质识别准确性
**难点**：用户主观回答可能影响准确性
**解决方案**：
- 使用经过验证的CCMQ量表
- 结合机器学习优化算法
- 收集用户反馈持续优化

### 8.2 兼夹体质识别
**难点**：用户可能同时具有多种体质特征
**解决方案**：
- 记录所有维度得分
- 突出显示主要体质和次要体质
- 提供综合调理方案

### 8.3 个性化推荐精准度
**难点**：用户个体差异大
**解决方案**：
- 收集用户反馈
- 使用协同过滤推荐算法
- 结合季节、地域等因素
